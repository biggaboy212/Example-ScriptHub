### To Build

- Install [DarkLua](https://github.com/seaofvoices/darklua/releases/latest) via [Aftman](https://github.com/LPGhatguy/aftman/releases)

```batch
aftman add seaofvoices/darklua
```

- Press ***Ctrl + Shift + B*** to build
